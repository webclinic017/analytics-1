class MarkovChain:

View In Hierarchy

Finds probabilities for transitions Usage:

>>> mc = MarkovChain()
>>> for category in [1, 1, 1, 1, 2, 3, 1, 1, 1, 2, 3]:
>>>     mc.add_observation(category)
>>> print(mc.get_stats(depth=1))

Will show that 2->3 100%, 3->1 100%, 1->1 80%, 1->2 20%
